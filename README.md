# あなたはどんな画家タイプ？―日本画家編―

日本画家の性格診断テストです。いくつかの質問に答えることで、あなたに似た日本画家のタイプを診断します。

## 使い方

1. `index.html` をブラウザで開く
2. 「診断を始める」ボタンをクリック
3. 5つの質問に答える
4. 診断結果が表示される
5. 代表作品のリンクをクリックすると、Google画像検索で作品を見ることができる
6. 「もう一度診断する」ボタンで最初から診断をやり直せる

## 仕組み

### ファイル構成

```
.
├── index.html    # メインページ（HTML構造）
├── style.css     # スタイルシート（デザイン）
├── script.js     # 診断ロジック（JavaScript）
└── README.md     # このファイル
```

### 診断の流れ

1. **質問ツリー構造**
   - 5階層の質問分岐で構成
   - 各質問には2つの選択肢がある
   - 最終的に11人の画家のいずれかにたどり着く

2. **質問の階層**
   - 第1層：伝統重視 vs 革新重視
   - 第2層：表現方法の選択
   - 第3層：題材や創作スタイルの選択
   - 第4層：細かい価値観の選択
   - 第5層：最終結果（画家タイプ決定）

3. **結果表示**
   - 画家の名前とアイコン
   - 代表作品（クリック可能なリンク）
   - 作品評価
   - 性格の説明

### 技術的な特徴

#### HTML (index.html)
- 3つの画面で構成
  - スタート画面
  - 質問画面（プログレスバー付き）
  - 結果画面

#### CSS (style.css)
- グラデーション背景
- レスポンシブデザイン対応
- ホバーエフェクト
- アニメーション効果
- モバイル対応

#### JavaScript (script.js)
- **画家データ（artists オブジェクト）**
  - 各画家の情報を格納
  - 名前、アイコン、代表作品、評価、性格
  - 作品検索用のクエリ

- **質問ツリー（questionTree オブジェクト）**
  - ネストされた構造で質問を管理
  - 各ノードに質問文と選択肢を持つ
  - 最終ノードに結果（画家キー）を持つ

- **主要な関数**
  - `startTest()` - 診断開始
  - `displayQuestion()` - 質問を表示
  - `selectOption()` - 選択肢を選んで次へ進む
  - `showResult()` - 結果を表示
  - `restartTest()` - 診断をリセット

## カスタマイズ方法

### 画家を追加する場合

1. `script.js` の `artists` オブジェクトに新しい画家を追加
2. `questionTree` に新しい分岐を追加

```javascript
newArtist: {
    name: "画家名",
    icon: "🎨",
    masterpiece: "代表作：『作品名』",
    works: [
        { title: "作品名", searchQuery: "画家名 作品名" }
    ],
    evaluation: "作品評価の説明",
    personality: "性格の説明"
}
```

### 質問を変更する場合

`questionTree` オブジェクトの `question` と `options` を編集します。

### デザインを変更する場合

`style.css` のカラーコードやフォントサイズを変更します。

主なカラー：
- メインカラー：`#667eea` → `#764ba2`（グラデーション）
- 背景色：白 `#ffffff`
- テキスト：`#333333`
