# あなたはどんな芸術家タイプ？

芸術家や絵画グループの性格診断テストです。いくつかの質問に答えることで、あなたに似た芸術家や絵画グループを診断します。

## 使い方

1. `index.html` をブラウザで開く
2. 診断したい芸術家グループを選択
   - 日本画家編
   - 印象派編
   - 浮世絵編
   - 世界の画家編
   - 絵画グループ編
   - アバンギャルド編
   - コンテンポラリー編
   - 世界の彫刻家編
   - クラシック音楽家編
   - 世界の文豪編
   - 日本の文豪編
3. 質問に答える（全グループ7階層）
4. 診断結果が表示される
5. 代表作品や代表画家のリンクをクリックすると、Google画像検索で作品を見ることができる
6. 「もう一度診断する」ボタンで同じグループを再診断
7. 「別のグループを診断する」ボタンでグループ選択画面に戻る

## 仕組み

### ファイル構成

```
.
├── index.html    # メインページ（HTML構造）
├── style.css     # スタイルシート（デザイン）
├── script.js     # 診断ロジック（JavaScript）
└── README.md     # このファイル
```

### 診断の流れ

1. **グループ選択**
   - 8つのグループから選択
   - 各グループに専用のアイコンと説明

2. **質問ツリー構造**
   - 全グループ7階層の質問分岐
   - 各質問には2つの選択肢がある
   - 最終的に診断結果にたどり着く
   - より詳細で正確な診断が可能

3. **結果表示**
   - **画家・彫刻家・音楽家・文豪診断の場合**
     - 芸術家の名前とアイコン
     - 代表作品（クリック可能なリンク）
     - 作品評価
     - 性格の説明
   - **絵画グループ診断の場合**
     - グループ名とアイコン
     - 概要と特徴
     - 代表的な画家（クリック可能なリンク）
     - 詳細情報ページへのリンク（Wikipedia）

### 技術的な特徴

#### HTML (index.html)
- 4つの画面で構成
  - グループ選択画面（11グループ）
  - スタート画面
  - 質問画面（プログレスバー付き）
  - 結果画面

#### CSS (style.css)
- グラデーション背景
- レスポンシブデザイン対応
- ホバーエフェクト
- アニメーション効果
- モバイル対応
- グループ選択ボタンのスタイリング

#### JavaScript (script.js)
- **グループ情報（groupInfo オブジェクト）**
  - 各グループのタイトルと説明

- **芸術家データ（各グループ用のオブジェクト）**
  - japaneseArtists - 日本画家データ（11人）
  - impressionismArtists - 印象派画家データ（10人）
  - ukiyoeArtists - 浮世絵師データ（11人）
  - worldArtists - 世界の画家データ（19人）
  - artMovements - 絵画グループデータ（15グループ）
  - avantgardeArtists - アバンギャルド画家データ（10人）
  - contemporaryArtists - コンテンポラリーアーティストデータ（10人）
  - sculptorArtists - 世界の彫刻家データ（10人）
  - musicianArtists - クラシック音楽家データ（10人）
  - writerArtists - 世界の文豪データ（10人）
  - japaneseWriterArtists - 日本の文豪データ（10人）
  - 各芸術家/グループの情報を格納
  - 名前、アイコン、代表作品、評価、性格/特徴

- **質問ツリー（各グループ用のオブジェクト）**
  - japaneseQuestionTree - 日本画家用（7階層）
  - impressionismQuestionTree - 印象派用（7階層）
  - ukiyoeQuestionTree - 浮世絵用（7階層）
  - worldQuestionTree - 世界の画家用（7階層）
  - artMovementQuestionTree - 絵画グループ用（7階層）
  - avantgardeQuestionTree - アバンギャルド用（7階層）
  - contemporaryQuestionTree - コンテンポラリー用（7階層）
  - sculptorQuestionTree - 世界の彫刻家用（7階層）
  - musicianQuestionTree - クラシック音楽家用（7階層）
  - writerQuestionTree - 世界の文豪用（7階層）
  - japaneseWriterQuestionTree - 日本の文豪用（7階層）
  - ネストされた構造で質問を管理
  - 各ノードに質問文と選択肢を持つ
  - 最終ノードに結果（芸術家/グループキー）を持つ

- **主要な関数**
  - `selectGroup()` - グループを選択
  - `backToGroupSelect()` - グループ選択画面に戻る
  - `startTest()` - 診断開始
  - `displayQuestion()` - 質問を表示
  - `selectOption()` - 選択肢を選んで次へ進む
  - `showResult()` - 結果を表示（画家/グループで表示内容を切り替え）
  - `restartTest()` - 診断をリセット
  - `showScreen()` - 画面切り替え

## 特徴的な機能

### グループ選択システム
- 11の異なる診断グループ
- 各グループに専用の質問ツリー
- 全グループ統一の7階層構造でより正確な診断

### 作品リンク機能
- 代表作品名をクリックでGoogle画像検索
- 代表画家名をクリックでGoogle画像検索（絵画グループ編）
- 新しいタブで開く

### 詳細情報リンク（絵画グループ編）
- Wikipediaへのリンク
- 各絵画グループの詳細を学べる

### プログレスバー
- 現在の進捗を視覚的に表示
- 全グループ7階層で統一された進捗表示

### レスポンシブデザイン
- PC、タブレット、スマートフォンに対応
- 画面サイズに応じてレイアウトが変化

## カスタマイズ方法

### 画家を追加する場合

1. `script.js` の該当する `artists` オブジェクトに新しい画家を追加

```javascript
newArtist: {
    name: "画家名",
    icon: "🎨",
    masterpiece: "代表作：『作品名』",
    works: [
        { title: "作品名", searchQuery: "画家名 作品名" }
    ],
    evaluation: "作品評価の説明",
    personality: "性格の説明"
}
```

2. 該当する `questionTree` に新しい分岐を追加

### 絵画グループを追加する場合

1. `script.js` の `artMovements` オブジェクトに新しいグループを追加

```javascript
newMovement: {
    name: "グループ名",
    icon: "🎨",
    description: "概要説明",
    characteristics: "特徴の説明",
    artists: [
        { name: "画家名", searchQuery: "画家名 paintings" }
    ],
    detailLink: "https://ja.wikipedia.org/wiki/グループ名"
}
```

2. `artMovementQuestionTree` に新しい分岐を追加

### 新しい診断グループを追加する場合

1. `groupInfo` に新しいグループ情報を追加
2. 画家/グループデータオブジェクトを作成
3. 質問ツリーを作成
4. `index.html` のグループ選択画面にボタンを追加
5. `startTest()` と `showResult()` 関数に条件分岐を追加

### デザインを変更する場合

`style.css` のカラーコードやフォントサイズを変更します。

主なカラー：
- メインカラー：`#667eea` → `#764ba2`（グラデーション）
- 背景色：白 `#ffffff`
- テキスト：`#333333`
- リンク：`#667eea`
